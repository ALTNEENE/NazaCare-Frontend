# NazaCare - Dengue Diagnosis

## Description

NazaCare is a web application that helps diagnose dengue fever using machine learning. It allows users to input their symptoms and get a prediction of whether they have dengue fever or not. It also provides information about dengue fever and how to prevent it.

## Features

- Symptom checker
- Dengue fever information
- Dengue fever prevention

- FastAPI

- PostgreSQL

## Installation

```bash
# Clone the repository
git clone <repository-url>

# Navigate to the frontend directory
cd frontend

# Install dependencies
npm install

# Run the development server
npm run dev
```

## Usage

Open [http://localhost:5173](http://localhost:5173) to view the application in your browser.
